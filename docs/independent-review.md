1. **Mechanical layer (CJK removal):** After Wave 2, every Tier 1 file must be free of CJK characters so the guardian, when promoted to a blocking gate in Wave 4, does not block any existing Tier 1 file.
2. **Principle layer (no hard-coded user-facing strings):** CLAUDE.md's Documentation Language Convention states that Tier 1 files "must not embed literal user-facing strings in any specific language. Any English literal that survives as a user-facing output string in a Tier 1 file is still a violation of this rule — the rule is language-agnostic.
